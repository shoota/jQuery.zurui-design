zurui line!

Drawing "awesome line", that appears to be a dig stacked the line of black and white.

.zuruiLine(position, opacity)

  • position(optional)

    Set "left" or "right" or "top" or "bottom" (as default).

  • opacity(optional)

    lineOpacity: solid line opacity. set between 0.1 and 1.0 (default: 0.2).
    shadowOpacity: line shadow opacity. set between 0.1 and 1.0 (default: 1.0).

zuruiLine();

zuruiLine("left");

zuruiLine("top", { lineOpacity: "0.1" });

zuruiLine("bottom", { lineOpacity: "1", shadowOpacity:"0.1" });

zurui text-shadow!

Decorate text that it is carved a seal.

.zuruiText(config)

  • config(optional)

    bgType: background color type. set "dark" when background-color is dark color.
    opacity: shadow opacity. set between 0.1 and 1.0 (default: 1.0).

$(selector).zuruiText();

$(selector).zuruiText({opacity: "0.5"});

$(selector).zuruiText({bgType: "dark"});

$(selector).zuruiText({bgType: "dark", opacity: "0.5"})

zurui gradation!

Make vertical gradation easy on the eyes.

.zuruiGradient(hexColor, percent)

  • hexColor(required)

    begin color of gradient.

  • percent(optional)

    gradient ratio[%] (default: 20%).
: Choose begin color

$(selector).zuruiGradient("#e6e4ce");

$(selector).zuruiGradient("#e6e4ce", 30);

$(selector).zuruiGradient("#e6e4ce", -30);

zurui radius! (Kado-Maru)

Set Radius to Box without/with simply animation.

.zuruiRadius(RAD, start, speed)

  • RAD(optional)

    Set radius[px] (default: 2[px]).

  • start(optional)

    If use animation, begin radius with this[px](default: RAD * 3 [px]).

  • speed(optional)

    If use animation, change rad[px]/sec. Set "slow" or "middle" or "fast", or Int value.

$(selector).zuruiRadius()

Auto set bender prefix. (powerd by jQuery v.1.8 or later)

$(selector).zuruiRadius(4, 12, 'middle')

zurui boss!

Awesome box with deboss/emboss effect. It is also good to use with texture background (try hover on each box)

.zuruiBoss(type, config)

  • type(optional)

    Select "deboss"(as deafult) or "emboss".

  • config(optional)

    border: solid line opacity. set between 0.1 and 1.0 (default: 0.1).
    shadow: line shadow opacity. set between 0.1 and 1.0 (default: 0.1).
    highlight: line highlight opacity. set between 0.1 and 1.0 (default: 1.0).

$(selector).zuruiBoss("deboss", {"border": "0.15", "shadow":"0.2", "highlight": "0.7"});

Deboss Box.

$(selector).zuruiBoss("emboss", {"border": "0.1", "shadow":"0.2", "highlight": "0.4"});

Emboss Box.
