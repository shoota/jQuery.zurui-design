jQuery.zurui-design

zurui line!

Default

Simple Usage as Default value(lineOpacity: "0.2",boxOpacity: "1").
$(selector).zuruiLine();

Set Position

Set Line Position. Select args in "top", "left", "right", or "bottom"(as default).
$(selector).zuruiLine("left");

Border Opacity

Change border Opacity as second arg object. Compare to this and 'default'.
$(selector).zuruiLine("top", { lineOpacity: "0.1" });

Box Shadow Opacity

Change box-shadow Opacity as second arg object.
$(selector).zuruiLine("bottom", { lineOpacity: "1", boxOpacity:"0.1" });

zurui text-shadow!

Default

Simple Usage as Default. When Backgroud color is lightness. Shadow Opacity is 1.0.
$(selector).zuruiText();

Set Shadow Opacity

Set Only Opacity.
$(selector).zuruiText({opacity: "0.4"});

Darkness background

When Backgroud color is darkness.
$(selector)..zuruiText({bgType: "dark"});

Darkness background and set Shadow Opacity

When Backgroun color is darkness and case of change Opacity.
$(selector).zuruiText({bgType: "dark", opacity: "0.5"})

zurui gradation!

Choose begin color:

Default (Lighten)

Simple Usage as Default value (i.e. percent:0.2). To lighten.
$(selector).zuruiGradient("#48b1f2");

Set Gradient value.

Generate Gradation To lighten with customize gradient val.
$(selector).zuruiGradient("#48b1f2", 0.1);

Darken

Generate Gradation To darken with customize gradient val.
$(selector).zuruiGradient("#48b1f2", -0.1);

zurui radius! (Kado-Maru)

Auto set bender prefix. (powerd by jQuery < v.1.8)

Generate Gradation To darken with customize gradient val.
$(selector).zuruiRadius(2);

With Animate.

Click here.

Draw radius with animate. zuruiRadius(endRad, beginRad, speed). speed can choose 'slow'/'fast'/'middle' or number val.
$(selector).zuruiRadius(4, 12, 'middle');

zurui box!

Inner Shadow Box.

Draw inner shadow.
$(selector).zuruiBox("inner");

Outer shadow Box.

Draw outer shadow.
$(selector).zuruiBox("outer");

Deboss Box.

Draw Deboss box.
$(selector).zuruiBoss("deboss", {"border": "0.15", "shadow":"0.2", "highlight": "0.7"});

Emboss Box.

Draw Emboss box.
$(selector).zuruiBoss("emboss", {"border": "0.1", "shadow":"0.2", "highlight": "0.4"});
